BETYdb

This is the source code for the Biofuel Ecophysiological Traits and Yields database (BETYdb)

The website is primarily written in Ruby-on-Rails, and has a MySQL backend. BETYdb provides an interface for contributing and accessing data, and is the informatics backend for the Predictive Ecosystem Analyzer (PEcAn).

See the wiki pages for documentation.
